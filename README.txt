EECS-481-HW-2
=============

Tic Tac Toe Game. Began with some of the code in the zip file on CTools and added checking  for a winner (which displays when there is a winner) and a new game button that clears the board.
To run the game open the package in eclipse and run on an android emulator.

Repository holding code for short android application built for EECS 481 homework assignment 2
